# Jumbotron

A Jumbotron indicates a big box for calling extra attention to some special content or information.

Find the code for this page in the `src/pages/components/Jumbotron.mdx` file.

## Props Reference

### Common Props
- `variant`: 'default' | 'hero' | 'post' | 'post-split' (default: 'default')
- `title`: string
- `image`: string (optional)

### Default & Hero Props
- `small`: boolean (optional)
- `description`: string (optional)
- `info`: string (optional)

### Post Variants Props
- `date`: string
- `author`: Object with properties:
  - `firstName`: string
  - `name`: string
- `categories`: Array of objects with properties:
  - `name`: string
  - `uri`: string

### Slots
- `intro`: Custom intro content
- `subtitle`: Subtitle content
- `cta`: CTA wrapper (default variant only)
- `cta-content`: CTA content (default variant only)
