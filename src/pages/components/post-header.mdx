# PostHeader

WordPress PostHeader - post top component for WordPress Single Post template data like categories etc. comes from WP Rest API.


## PostHeader with some content for WordPress single page template

<div class="component-preview">
  <div class="relative w-full">
    <PostHeader
      author={author}
      lang="en"
      categories={categories}
      title="Lorem Ipsum"
      date={date}
      image="https://placehold.co/1380x920"
    />
  </div>
</div>

```js
  <div class="relative">
    <PostHeader
      author={ { firstName: 'Szymon', name: 'CNK' }  }
      lang="en"
      categories={  [{"name":"Tools & garage accessories","uri":"/category/garage/tools-garage-accessories/"}, , {"name":"Lorem Ipsum","uri":"#"}] }
            title="Lorem Ipsum"
      date="2023-07-13T01:35:03"
      image="https://placehold.co/1380x920"
    />
  </div>
```

## Props

<PropsTable data={[
  { name: 'title', type: 'string', required: true, description: 'Post title displayed as the main heading.' },
  { name: 'image', type: 'string', required: false, description: 'Hero image URL displayed behind/above the title.' },
  { name: 'lang', type: 'string', required: false, description: 'Language code used by date formatting (e.g. `en`, `pl`).' },
  { name: 'date', type: 'string', required: false, description: 'ISO 8601 publication date (e.g. `2023-07-13T01:35:03`).' },
  { name: 'author', type: '{ firstName: string; name: string }', required: false, description: 'Post author — `name` is displayed, `firstName` populates the title attribute.' },
  { name: 'categories', type: '{ name: string; uri: string }[]', required: false, description: 'Category objects rendered as links above the title.' },
]} />
